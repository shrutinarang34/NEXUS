# **App Name**: Expense Tracker

## Core Features:

- User Authentication: User Authentication: Implement user sign-up and login using Firebase Auth.
- Expense Logging: Expense Logging: Log expenses with amount, description, date, category, and account (optional).
- Expense List: Display Expense list using Nextjs.
- Customizable Settings: Customizable Settings: Add, edit, or delete custom expense categories and accounts.
- Dashboard total: Display date range expense total using Nextjs.
- Filtering: Filtering: Filter dashboard charts and expense list by date range.
- CSV Export: CSV Export: Export filtered expense data to a CSV file.

## Style Guidelines:

- Primary color: Deep violet (#9400D3) for calmness, yet remains sophisticated and modern.
- Background color: Very light violet (#F2E7FF), offering a clean and subtle backdrop.
- Accent color: Purple (#9F00FF), bringing contrast and emphasizing key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, neutral, and readable user interface.
- Use minimalist icons to represent expense categories.
- Fully responsive layout that adapts to different screen sizes.
- Subtle animations and transitions for a smooth user experience.
